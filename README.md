Map-Based UI Application developed using Swift and SwiftUI with MVVM architecture. The app supports 
light and dark modes, adapts to iPhone and iPad in both orientations, and features 
interactive custom map pins with animations and a dynamic menu for location details.
Technologies Used: Swift, SwiftUI, MVVM architecture
Platform: iOS (iPhone and iPad)
Features:
Adaptive UI for light and dark modes
Support for both portrait and landscape orientations
Interactive map with custom pins and animations
Dynamic menu displaying location details with seamless navigation
Efficient and readable code structure, maintaining neat and organized architecture
